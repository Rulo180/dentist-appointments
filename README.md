# dentist-appointments

Web application for doctors to allow their patients to schedule appointments online. It also allows you to keep a medical history of patients to monitor their health.
This app is built using Meteor + React

[![Build Status](https://travis-ci.org/Rulo180/dentist-appointments.svg?branch=master)](https://travis-ci.org/Rulo180/dentist-appointments)

## Features
- User authentication
- Social secures CRUD
- Patients CRUD
- Appointments CRUD
- Appointment confirmation/cancellation
- Internationalization for english and spanish

## Usage

To run the app you have to use this command:
```
npm start
```

## Technologies

- Meteor
- React
- Bootstrap
- SASS
- MongoDB
